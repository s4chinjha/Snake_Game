# 🐍 Snake Game in Java

A classic Snake Game developed using Java.  
This project demonstrates fundamental game development concepts such as game loops, keyboard input handling, collision detection, score tracking, and object-oriented programming.

## 🎮 Features

- Classic Snake gameplay
- Smooth snake movement
- Food spawning system
- Score tracking
- Collision detection
- Game Over screen
- Simple and clean user interface

## 🛠️ Tech Stack

- Java
- Java Swing
- Java AWT

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- Java JDK 8 or above
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code, etc.)

## 📥 Clone the Repository

```bash
git clone https://github.com/s4chinjha/Snake_Game.git
```

## ▶️ Run the Project

### Using Terminal

```bash
javac Main.java
java Main
```

### Using an IDE

1. Open the project in your preferred IDE
2. Compile and run the main Java file

## 🎯 Controls

| Key | Action |
|------|--------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |
